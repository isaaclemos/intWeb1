Tutorial de instalação do LibreOffice no Debian

O LibreOffice 6.0 vem com muitas novidades e melhorias! Atualize facilmente seu LibreOffice 5.0 para 6.0 e confira os novos recursos e funcionalidades.
Não sabe qual é a versão do seu LibreOffice? Faça o seguinte: abra o LibreOffice Writer, clique no menu "Ajuda" e "Sobre o LibreOffice".

No dia 31 de janeiro de 2018, foi lançada a versão 6.0 do LibreOffice, versão esta que trouxe vários novos recursos e melhorias adicionados aos seus componentes.
Dentre algumas das novidades estão o recurso de assinatura e criptografia de documentos com OpenPGP, novas funções de planilha para o Calc, um novo estilo de tabela padrão do Writer, a possibilidade de exportar os documentos para o formato EPUB3, inclusão dos ícones do elementary OS e duas novas variantes da "Notebookbar".
O LibreOffice é uma poderosa suíte de produtividade de escritório de código aberto com planilha, editor de texto, editor de apresentação e muito mais. É destinado tanto para uso pessoal quanto comercial e disponível gratuitamente para Linux, Windows e macOS. Com ele, você pode abrir arquivos criados com os aplicativos Word, Excel, Power Point, Access e Corel Draw, além de montar documentos compatíveis com esses formatos.

Acompanhe abaixo como instalar o LibreOffice 6.0 no Ubuntu 16.04 LTS ou superior

1)- Abra uma nova janela do Terminal (Ctrl+Alt+T) e adicione o PPA do LibreOffice 6.0, digitando o comando abaixo:

sudo add-apt-repository ppa: libreoffice / ppa

2)- Atualize o sistema para atualizar a lista de pacotes disponíveis, digitando o seguinte comando abaixo:

sudo apt update

3)- Digite o comando abaixo para instalar ou atualizar o LibreOffice 6.0:

sudo apt install libreoffice

Pronto! Agora abra o LibreOffice e verifique se sua atualização foi feita com sucesso (opção Ajuda > Sobre).
